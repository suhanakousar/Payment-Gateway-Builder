🏦 STEP 1: CREATE VENDOR (SUB-MERCHANT)

On KYC approval, call Cashfree:

POST /pg/easy-split/vendors

Body:
{
"vendor_id": "<merchant_id>",
"name": "<merchant_name>",
"email": "<merchant_email>",
"phone": "<merchant_phone>",
"bank": {
"account_number": "<bank_account>",
"ifsc": "<ifsc>",
"account_holder": "<holder_name>"
}
}

Store in DB:

* merchants.provider_merchant_id = vendor_id
* merchants.provider_status = PENDING

---

🔄 STEP 2: VENDOR STATUS SYNC (CRON)

Every 2 minutes:

GET /pg/easy-split/vendors/{vendor_id}

If status == ACTIVE:
merchants.provider_status = ACTIVE

If status == REJECTED:
merchants.provider_status = REJECTED

---

⛔ STEP 3: BLOCK ORDER IF NOT ACTIVE

Before creating order:

IF provider_status != ACTIVE:
return HTTP 412 "Merchant not ready for payments"

---

💳 STEP 4: CREATE ORDER WITH SPLIT

Modify providers/cashfree.ts:

POST /pg/orders

Body:
{
"order_id": "<orderId>",
"order_amount": <amount>,
"order_currency": "INR",
"customer_details": {
"customer_id": "<id>",
"customer_phone": "9999999999"
},
"order_splits": [
{
"vendor_id": "<merchant.provider_merchant_id>",
"percentage": 100
}
]
}

Store:
orders.txn_id = cf_order_id

---

📱 STEP 5: GENERATE UPI QR

POST /pg/orders/pay

Body:
{
"order_id": "<orderId>",
"payment_method": {
"upi": {
"channel": "qrcode"
}
}
}

Response:

* upi_intent or payment_link

Generate QR:
qrImage = generateQRCode(upi_intent)

Return to frontend:
{
qrImage,
txnId
}

🔔 STEP 7: WEBHOOK (CRITICAL)

POST /api/webhook?provider=cashfree
